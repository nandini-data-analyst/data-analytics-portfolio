# Excel Sales Dashboard

## Project Overview
This project analyzes product sales using an Excel dashboard. It provides insights into sales performance by product, location, and month.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Key Analysis
- Total sales amount
- Sales by products
- Sales by locations
- Monthly sales trends
