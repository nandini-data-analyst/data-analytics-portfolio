# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data to identify trends in revenue, profit, and customer behavior.

## Tools Used
- Excel
- Power BI
- SQL

## Key Analysis
- Sales performance by region
- Profit trends by product category
- Impact of discounts on profit

## Insights
- Identified top performing product categories
- Found regions with highest profit margins
- Analyzed customer purchase patterns
