# Blinkit Sales Dashboard

## Project Overview
This project analyzes Blinkit retail sales data to understand product performance, outlet performance, and sales trends.

## Tools Used
- Power BI
- Excel

## Key Analysis
- Total sales and average sales
- Sales by outlet location and outlet size
- Sales by item type
- Product category performance

## Insights
- Identified top selling product categories
- Compared outlet performance by location
- Analyzed sales trends across different outlet types
